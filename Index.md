# Chart & Infographic Library

A comprehensive reference library for data visualization — chart types, infographic formats, visual styles, data contexts, and use cases. Focused on **energy markets, climate change, finance, sustainability, and the global energy transition**.

---

## Chart Types

### Comparison
- [[Bar Chart]] — horizontal bars comparing categories
- [[Column Chart]] — vertical bars comparing categories
- [[Lollipop Chart]] — dot-on-stick for clean comparisons
- [[Bullet Chart]] — actual vs target performance
- [[Dot Plot]] — individual values on a scale
- [[Dumbbell Chart]] — paired before/after comparisons
- [[Radar Chart]] — multivariate profile comparison
- [[Tornado Chart]] — back-to-back bars for two-group/scenario comparison
- [[Quadrant Chart]] — scatter plot divided into 4 labeled zones
- [[Radial Bar Chart]] — circular bar layout for cyclical data

### Distribution
- [[Histogram]] — frequency distribution of continuous data
- [[Box Plot]] — five-number summary across groups
- [[Violin Plot]] — full distribution shape + summary stats
- [[Density Plot]] — smooth probability distribution curve
- [[Ridgeline Plot]] — overlapping distributions over time
- [[Population Pyramid]] — mirrored bars for two-group demographic comparison

### Composition (Part-to-Whole)
- [[Pie Chart]] — simple proportions (2-4 categories)
- [[Donut Chart]] — proportions with center annotation
- [[Treemap]] — hierarchical nested rectangles
- [[Sunburst Chart]] — hierarchical concentric rings
- [[Waffle Chart]] — grid-based percentage display
- [[Stacked Bar Chart]] — composition within categories
- [[Marimekko Chart]] — two-variable mosaic
- [[Pictogram Chart]] — icon-based proportional display
- [[Nightingale Rose Chart]] — polar area with varying radius

### Hierarchical
- [[Packed Circle Chart]] — nested circles showing hierarchy

### Relationship
- [[Scatter Plot]] — two-variable correlation
- [[Bubble Chart]] — three-variable correlation
- [[Heatmap]] — intensity across two dimensions
- [[Correlogram]] — multi-variable correlation matrix
- [[Network Diagram]] — nodes and connections
- [[Parallel Coordinates]] — multi-axis multivariate comparison
- [[Connected Scatter Plot]] — sequential trajectory in 2D space

### Flow
- [[Sankey Diagram]] — proportional flow between categories
- [[Alluvial Diagram]] — flows across time stages
- [[Chord Diagram]] — bidirectional circular flows
- [[Funnel Chart]] — sequential stage reduction

### Temporal
- [[Line Chart]] — trends over time
- [[Area Chart]] — magnitude trends over time
- [[Stream Graph]] — flowing composition over time
- [[Bump Chart]] — ranking changes over time
- [[Candlestick Chart]] — financial OHLC data
- [[Gantt Chart]] — project timelines and milestones
- [[Racing Bar Chart]] — animated rank changes over time
- [[Sparkline]] — word-sized inline trend indicator

### Geospatial
- [[Choropleth Map]] — regions colored by value

- [[Tile Map]] — equal-sized tiles eliminating geographic bias

### KPI & Dashboard

## Quick Reference

> [!tip] How to Choose a Chart
> 1. **What data do you have?** Categorical, continuous, time series, geographic, hierarchical, relational?
> 2. **What do you want to show?** Compare, trend, distribute, compose, relate, flow?
> 3. **Who is your audience?** General public, policymaker, investor, researcher, designer?
> 4. **Where will it appear?** Print, screen, social media, dashboard, presentation?
