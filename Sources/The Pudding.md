# The Pudding

## Overview
The Pudding is a digital publication that creates visual essays exploring ideas debated in culture. Founded in 2017, they push the boundaries of what data visualization can be, combining rigorous data analysis with playful, often gamified interactive experiences. Their work is entirely web-native, treating the browser as a creative medium rather than just a distribution channel.

## Signature Visual Style
- **Color palette**: Bright, saturated, playful. No fixed brand palette — each story gets a custom color scheme matching its tone. Common patterns: neon accents on dark backgrounds, candy-colored categories, bold gradients
- **Typography**: Large, bold sans-serif headlines (often custom or display fonts per story). Conversational body text. Typography treated as a design element, not just a text container
- **Layout**: Full-viewport scroll-driven experiences. Immersive — no chrome, no sidebar, no standard page template. Each essay is a custom-designed experience
- **Tone**: Playful, curious, irreverent. "Serious analysis, unserious presentation"

## Strengths
- Pushing the frontier of what interactive data visualization can be on the web
- User-input data stories where the reader's own data becomes part of the visualization
- Exceptional craft in scroll-driven animations and transitions using D3.js and Svelte
- Cultural data topics (music, language, film, sports) that expand the audience for data viz
- Transparent methodology with detailed write-ups of data collection and analysis

## Chart Types They Favor
- [[Custom Interactive Chart]] — bespoke visualizations invented for each story, defying standard chart taxonomy
- [[Racing Bar Chart]] — animated bar charts showing rankings change over time
- [[Connected Scatter Plot]] — showing trajectory through two-variable space over time
- [[Network Diagram]] — relationship mapping between cultural entities (artists, genres, influences)
- [[Scroll-Driven Animation]] — charts that morph, build, and transform as the reader scrolls through narrative

## Design Patterns Worth Studying
- **User-input data stories**: Asking readers to input their own data (location, preferences, demographics) and then showing how they compare to the dataset — creating personal investment in the story
- **Gamified exploration**: Turning data exploration into quiz-like or game-like interactions where readers make predictions before seeing the answer
- **Methodology transparency**: Every essay includes a detailed methodology section explaining data collection, cleaning, and analytical choices
- **Svelte + D3 hybrid**: Using Svelte for reactive UI and D3 for data manipulation and SVG generation, achieving smooth performance for complex animations

## Energy & Climate Relevance
The Pudding rarely covers energy or climate directly, but their interactive techniques are highly transferable. Their methods for making abstract data personal (through user input and comparison), gamifying exploration, and building scroll-driven narratives could transform how climate and emissions data is communicated to general audiences.

## Key Reports / Products
- **Visual essays** — Published monthly, each a standalone interactive experience on a cultural topic
- **The Pudding Cup** — Annual award celebrating the best visual and data stories from other publications
- **Open-source tools** — Shared components and templates for scroll-driven storytelling
- **Methodology write-ups** — Detailed behind-the-scenes documentation of data and design decisions
- **Svelte starter templates** — Reusable project scaffolds for interactive data stories

## Data Access
- All essays freely viewable on the web, no paywall
- Datasets sometimes shared on GitHub alongside methodology documentation
- Code repositories occasionally open-sourced for notable projects
- No formal data portal or API
- Creative Commons licensing on selected assets and datasets
