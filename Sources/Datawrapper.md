# Datawrapper

## Overview
Datawrapper is a chart creation tool and data visualization platform used by newsrooms, researchers, and organizations worldwide. Founded in 2012 in Berlin, it enables non-designers to create editorial-quality responsive charts with minimal effort. Beyond the tool itself, Datawrapper has built a vibrant community and blog that has become one of the best resources for chart design thinking and best practices.

## Signature Visual Style
- **Color palette**: Clean, editorial defaults. Accessible color palettes tested for colorblindness. Grey (#CCCCCC) for secondary data, single highlight color for emphasis. Default palette: warm grey base with selective blue, red, or green accents
- **Typography**: System-optimized sans-serif (Roboto, Open Sans). Clear axis labels, generous tick spacing. Annotation text in slightly smaller size with muted color
- **Layout**: Responsive by default — charts automatically adapt from desktop to mobile with layout adjustments (not just scaling). Annotation repositioning at breakpoints. Embeddable iframe format
- **Tone**: Helpful, educational, democratizing. "Good charts for everyone" philosophy

## Strengths
- Lowest barrier to entry for creating publication-quality responsive data visualizations
- Responsive design as a core feature, not an afterthought — charts genuinely work on every screen size
- Exceptional documentation and blog with chart design tutorials, color theory, and accessibility guides
- River community gallery showcasing how real organizations use Datawrapper for inspiration
- Annotation tools that make it easy to add context, highlights, and direct labels to any chart

## Chart Types They Favor
- [[Line Chart]] — their most-used chart type, with annotation tools for highlighting trends and events
- [[Bar Chart]] — horizontal and vertical, with built-in sorting, highlighting, and direct labeling
- [[Choropleth Map]] — extensive map library with country, state, and custom boundary support
- [[Scatter Plot]] — with built-in trendlines, quadrant labels, and selective point labeling
- [[Small Multiples]] — their 2025 breakout feature, enabling grid comparisons with consistent scales

## Design Patterns Worth Studying
- **Responsive-first chart design**: Charts that genuinely restructure for mobile — axis labels rotate, annotations reposition, legends move from side to bottom — not just CSS scaling
- **Annotation as storytelling**: Built-in range annotations, reference lines, and callout labels that encourage every chart to tell a story rather than just display data
- **River community gallery**: Curated showcase of charts made by users worldwide, organized by chart type and topic — an endless source of real-world design inspiration
- **Blog as design education**: Detailed posts on topics like "What to consider when choosing colors," "When to use a log scale," and chart type selection guides

## Energy & Climate Relevance
Datawrapper is used by many energy and climate organizations (including OWID, Ember, and numerous newsrooms) to create and embed data visualizations. Their tool is directly relevant for creating GHG inventory dashboards, emissions trend charts, and stakeholder-facing visualizations without needing custom code. Excellent option for rapid prototyping of climate data presentations.

## Key Reports / Products
- **Datawrapper app** — Core chart creation tool with 20+ chart types, maps, and tables
- **Datawrapper Academy** — Tutorial library covering chart creation, design principles, and data preparation
- **Blog** — Weekly posts on data visualization design, color, typography, and chart selection
- **River** — Community gallery of published Datawrapper charts organized by type and topic
- **Custom design themes** — Enterprise feature for applying brand guidelines to all charts organization-wide

## Data Access
- Free tier available with basic features and Datawrapper branding
- Paid plans for custom branding, team collaboration, and advanced features
- Charts embeddable via iframe with responsive behavior
- Export to PNG/SVG for static use
- API available for programmatic chart creation and data updates
